# 🏦 Bank Account Management System

A simple **Bank Account Management System** built using **Python Object-Oriented Programming (OOP)** concepts. This project demonstrates encapsulation, private variables, private methods, getters, setters, exception handling, and file handling.

## 🚀 Features

* Create a bank account using user input
* Store account number and balance securely using private variables
* Validate deposit amounts
* Deposit money and update account balance
* Display account details
* Handle invalid inputs using exception handling
* Save transaction details automatically into `bank.txt`

## 🛠️ Concepts Used

* Python Classes & Objects
* Constructors (`__init__`)
* Encapsulation
* Private Variables
* Private Methods
* Getter Methods
* Data Validation
* Exception Handling
* File Handling

## ▶️ How to Run

1. Clone this repository:

```bash
git clone <your-repository-link>
```

2. Open the project folder:

```bash
cd Bank-Account-Management-System
```

3. Run the Python file:

```bash
python main.py
```
